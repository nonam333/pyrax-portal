# Pyrax Crypto News Portal Design Guidelines

## Design Approach
**Reference-Based Approach**: Inspired by theblock.co but elevated with stronger brand identity, enhanced visual hierarchy, and more engaging user interactions. The design prioritizes elegance, sophistication, and professional credibility while maintaining high user engagement.

## Core Design Elements

### A. Color Palette
**Primary Colors:**
- Vivid Orange Gradient: 20 100% 60% to 35 100% 55% (for CTAs, highlights, hover states)
- Pure Black: 0 0% 0% (primary background)
- White: 0 0% 100% (text content)

**Accent Colors:**
- Dark Gray: 0 0% 15% (card backgrounds, borders)
- Medium Gray: 0 0% 40% (secondary text, metadata)

### B. Typography
- **Primary Font**: Inter or similar modern sans-serif via Google Fonts
- **Headlines**: Bold weights (700-800) for impact
- **Body Text**: Regular (400) and medium (500) weights
- **Metadata**: Light (300) for timestamps, categories

### C. Layout System
**Tailwind Spacing Primitives**: 2, 4, 6, 8, 12, 16, 24
- Micro spacing: p-2, gap-2
- Standard spacing: p-4, m-4, gap-4
- Section spacing: p-8, mb-8
- Large spacing: p-16, mb-24

### D. Component Library

**Navigation:**
- Black background with Pyrax logo only
- Clean, minimal design
- Live crypto ticker tape at very top

**Hero Section:**
- Large featured article with prominent image
- Orange gradient overlay on image for text readability
- Bold headline typography
- Prominent CTA button with orange gradient

**News Grid:**
- Newspaper-style dense layout
- Card-based design with hover effects
- Orange accent borders on hover
- Category tags with orange gradient backgrounds

**Markets Components:**
- Crypto screener table with sortable columns
- Orange hover highlights on rows
- Live price updates with color-coded changes
- Search and filtering controls

**CTAs & Buttons:**
- Orange gradient backgrounds for primary actions
- Smooth hover transitions
- Newsletter signup blocks prominently placed
- Social sharing buttons in orange theme

### E. Content Strategy

**Homepage Sections (Maximum 5):**
1. Hero with featured article
2. Latest news grid (3-4 columns)
3. Markets overview widget
4. Newsletter signup block
5. Trending/Popular content section

**Rich Content Areas:**
- Multiple article cards per row
- Sidebar with crypto categories
- Ad placement zones between content sections
- Related articles sections
- Community/comments integration

**Individual Coin Pages:**
- Large coin logo and name header
- Live price chart integration
- Market data overview
- Related news feed below
- Community discussion section

### F. Performance & Responsiveness
- Mobile-first design approach
- Lazy loading for images and data
- Smooth animations limited to hover states and loading
- Fast data refresh for live prices
- Google AdSense integration zones
- SEO optimization throughout

## Images
**Hero Image**: Large, high-quality crypto/finance related imagery with orange gradient overlay for text contrast
**Article Images**: Professional crypto news photos, company logos, charts
**Coin Logos**: Official cryptocurrency brand logos in their original colors
**Background**: Subtle texture or pattern on black background for visual interest

**Image Placement:**
- Hero section: Full-width featured image
- News cards: Thumbnail images (16:9 ratio)
- Coin pages: Large coin logos prominently displayed
- Ad zones: Designated spaces between content sections

The design emphasizes content density like theblock.co while elevating the visual appeal through strategic use of the orange gradient, smooth interactions, and professional typography hierarchy.
